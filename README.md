###📊 Digital Marketing Performance – EDA & ROI Analysis

**📌 Project Overview**
This project performs Exploratory Data Analysis (EDA) on a digital marketing dataset to evaluate campaign performance across different platforms.
The objective is to analyze the marketing funnel, calculate Return on Investment (ROI), and recommend which platforms should receive more budget allocation based on profitability.

**🎯 Objective**
Clean and prepare the dataset

Analyze the marketing funnel (Impressions → Clicks → Conversions)

Calculate ROI for each platform

Compare revenue and spending

Provide budget recommendations based on performance

This project focuses on business insights rather than building a machine learning model.

**🗂 Dataset Used**
Digital Marketing Performance Dataset

The dataset includes the following key columns:
platform

spend

impressions

clicks

conversions

revenue

**🧹 Data Cleaning**
Removed missing values

Removed duplicate records

Converted numerical columns to proper numeric format

This ensured accurate calculations and reliable analysis.

**📊 Funnel Analysis**
The marketing funnel was analyzed using:

Total impressions

Total clicks

Total conversions

A significant drop was observed between impressions and clicks, indicating opportunities to improve ad engagement and targeting strategies.

**💰 ROI Analysis**
ROI was calculated using:

ROI = (Revenue − Spend) / Spend

Key Findings:
Highest ROI Platform: Google Search

Lowest ROI Platform: LinkedIn

Highest Revenue Platform: Google Search

Google Search delivers strong positive returns and generates the highest revenue. LinkedIn shows negative ROI, meaning spending exceeds revenue generated.

**📈 Business Recommendation**
Increase budget allocation for Google Search

Reduce or re-evaluate spending on LinkedIn

Optimize funnel stages to improve engagement and conversion rates

Allocating more budget to high-performing platforms will maximize overall profitability.

**🛠 Tools Used**
Python

Google Colab

Pandas

Matplotlib

Seaborn

**✅ Conclusion**
This analysis demonstrates that marketing performance varies significantly across platforms. By focusing on ROI-driven budget allocation and optimizing funnel performance, businesses can improve marketing efficiency and overall returns.
